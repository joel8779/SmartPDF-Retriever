📄 SmartPDF Retriever
AI-Powered PDF Question Answering System (RAG + FastAPI + Streamlit + Ollama + FAISS)

SmartPDF Retriever is a full-stack AI application that allows users to upload any PDF and ask questions about it.
The system uses:

Retrieval-Augmented Generation (RAG)

LangChain

FAISS Vector Store

HuggingFace Embeddings

Ollama LLM (Llama3 / Mistral / Gemma)

FastAPI Backend

Streamlit Frontend

Everything runs locally — no API costs, no internet data sharing, fully private.

🚀 Features

✔ Upload any PDF
✔ Extract & chunk text
✔ Generate embeddings (MiniLM)
✔ Store vectors in FAISS
✔ Query using semantic search
✔ LLM-powered answers using local Ollama
✔ Full-stack architecture (FastAPI + Streamlit)
✔ 100% offline and free
